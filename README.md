# ACEYA Youth Conference 2026

Quick-reference web app for the 38th Annual ACEYA Youth Conference — Detroit, MI, July 3–8, 2026.

## Edit content

Open `index.html` and update the `CONFERENCE_DATA` object near the top of the `<script>` block. No other files need to change.

## Local preview

Open `index.html` in a browser, or run:

```bash
npx serve .
```

## Deploy

Deployed on Vercel. Push changes to `main` to trigger a new deployment (if Git integration is enabled).
